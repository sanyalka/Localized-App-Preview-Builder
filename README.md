# App Preview Generator

Веб-приложение для массового создания локализованных превью приложений.

## Возможности

- Загрузка нескольких шаблонов (изображений)
- Загрузка JSON с переводами на любые языки
- Добавление неограниченного количества текстовых элементов
- Настройка шрифта, размера, цвета, тени, обводки, выравнивания
- Drag & drop перемещение текста прямо на canvas
- Предпросмотр результата для каждого языка
- Генерация ZIP архива со структурой папок по языкам

## Как пользоваться

1. Откройте `index.html` в браузере
2. Загрузите шаблоны (PNG/JPG/WEBP)
3. Загрузите JSON с переводами (см. пример ниже)
4. Добавьте текстовые элементы и укажите ключи из JSON
5. Перетаскивайте текст на canvas для точного позиционирования
6. Нажмите **Сгенерировать и скачать ZIP**

## Формат JSON

```json
{
    "en-UK": {
        "appName": "FileFort",
        "tagline": "PDF Redactor"
    },
    "ru-RU": {
        "appName": "FileFort",
        "tagline": "PDF редактор"
    },
    "de-DE": {
        "appName": "FileFort",
        "tagline": "PDF-Editor"
    }
}
```

Ключи (`appName`, `tagline`) используются в текстовых элементах конструктора.

## Структура выходного ZIP

```
app-previews.zip
├── en-UK/
│   ├── template1.png
│   └── template2.png
├── ru-RU/
│   ├── template1.png
│   └── template2.png
└── de-DE/
    ├── template1.png
    └── template2.png
```

## Технологии

- Vanilla JavaScript (без сборщиков)
- HTML5 Canvas
- JSZip + FileSaver.js
